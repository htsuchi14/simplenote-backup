# Simplenote ノート自動分類

## 概要

バックアップディレクトリ内の未分類ノートを**自動で連続分類**します。
確認なしで100件ずつ処理し、未分類がなくなるまで繰り返します。

## 既存タグ一覧

!`python3 simplenote-classify.py tags`

## 現在の状態確認

!`python3 simplenote-classify.py status`

## 自動処理ループ

**動的に対応**: 未分類ノートがなくなるまで以下を繰り返す

### Step 1: タグ付きファイルの整理

`has_tag_not_moved_count > 0` の場合、先に整理:
```bash
python3 simplenote-classify.py organize
```

### Step 2: 未分類ノートの分類

`needs_classification_count > 0` の場合:

1. `python3 simplenote-classify.py json` から未分類ノートを取得
2. 各ノートに適切なタグを判定
3. **確認なしで即座に** `python3 simplenote-classify.py apply` を並列実行（10件ずつ）
4. 100件処理したら残り件数を確認
5. 残りがあれば次の100件へ

### Step 3: 完了確認

`python3 simplenote-classify.py status` で確認:
- `root_files_total: 0` になれば完了

## タグ付けガイドライン

既存タグの用途:
- **思考**: 考え事、アイデア、哲学的メモ、自己分析
- **ネタ**: お笑い、ジョーク、コントのアイデア
- **仕事**: 業務関連、会議メモ、タスク、プロジェクト
- **プログラミング**: コード、技術メモ、API、開発関連
- **読書**: 本の感想、引用、書評、要約
- **ライフ**: 生活全般、買い物、契約、人名メモ、旅行
- **Health**: 健康、医療、運動記録
- **ゴルフ**: ゴルフ技術、スコア、練習メモ
- **エッセイ**: 長文の文章、随筆
- **コラム**: 意見、論評
- **ストーリー**: 物語、創作、小説
- **大喜利**: 大喜利のお題と回答
- **スケジュール**: 予定、カレンダー

判断に迷う場合は最も近いタグを選択。新規タグは作成しない。

## 実行時の出力

各バッチ完了時に簡潔に報告:
```
## バッチ 1 完了（100件）
仕事: 25件, ライフ: 20件, 思考: 18件, ...
残り: XXX件 → 次のバッチへ

## バッチ 2 完了（100件）
...
```

最終完了時:
```
## 全分類完了
総処理: XXX件（動的にカウント）
ルートの未分類: 0件
```
